# tetras
A small interactive sim/visualizer of flocking tetras. Uses flocking algorithm concepts like cohesion, alignment, and separation for tetra movement. You can adjust the pH and temperature of the environment, which will modify the flocking behavior of the tetras.

<img src="Media/tetrasGif_01.gif" width="400">


### Built With
- Unity 6000.0.58f2
- C#
- URP

### How to run
- Open project in Unity 6000.0.58f2 or later
- Press play from `Main` scene
- Use UI buttons in top left to adjust pH or temperature to modify flocking behaviors
